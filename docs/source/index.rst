COMPE 560: UDP-based chat application documentation
===================================================

A secure UDP‐based chat application built in Python for COMPE 560.  
It provides real‐time encrypted communication between multiple clients
via a central server, using hybrid RSA/AES encryption, HMAC authentication,
and a curses‐based terminal UI.
